🔍 施術前事前診断：16パターン特定フローチャート

クライアントの症状と生活習慣から、該当する「パターン番号」を特定するためのスクリーニング・マニュアルです。施術前のカウンセリングでご活用ください。

1症状分布の確認（日中の姿勢特定）

クライアントに「一番つらい場所」を2箇所挙げてもらい、その組み合わせから日中の姿勢タイプを特定します。

🗣️ 問診トーク例：
「上半身と下半身で、それぞれ一番つらい場所を1つずつ指差していただけますか？」
「右肩と左腰、のように対角線上に痛みがありますか？それとも片側に寄っていますか？」
TYPE A：クロス右
（雑巾絞り型）
右肩 × 左腰

ねじれストレス

  • 右脚を上にして組む癖がある
  • 車の運転が多い（右ハンドル）
  • 左の骨盤が下がっている感覚
  • 右肩が前に巻いている
TYPE B：クロス左
（雑巾絞り型・逆）
左肩 × 右腰

ねじれストレス

  • 左脚を上にして組む癖がある
  • PC画面がデスクの右側にある
  • 右の骨盤が下がっている感覚
  • 左肩が前に巻いている
TYPE C：ラテラル右
（右潰れ・アコーディオン型）
右肩 × 右腰

片側圧縮ストレス

  • 右肘をつく癖がある
  • 常に右肩で鞄を持つ
  • 立っている時、右足重心
  • 身体全体が右に傾いている
TYPE D：ラテラル左
（左潰れ・アコーディオン型）
左肩 × 左腰

片側圧縮ストレス

  • 左肘をつく癖がある（車のドア等）
  • 常に左肩で鞄を持つ
  • 立っている時、左足重心
  • 身体全体が左に傾いている

2睡眠姿勢の確認

4診断後のアクション指針

⚠️ ★最重症パターン（No.02, 06, 09, 13）の場合

日中に負担をかけている筋肉を、夜間も圧迫し続けています。「24時間休息なし」の状態であることを伝え、睡眠姿勢の変更（仰向け推奨）と抱き枕の使用を強く提案してください。

📋 施術方針の決定
  • TYPE A・B（クロス型）の場合： 背骨を中心とした「回旋（ねじれ）」を取る手技をメインにします。
    → 広背筋、脊柱起立筋の対角線ストレッチ
  • TYPE C・D（ラテラル型）の場合： 体側を中心とした「側屈（ちぢみ）」を取る手技をメインにします。
    → 腰方形筋、中殿筋、脇腹のリリース
  • うつ伏せ寝（No.04, 08, 12, 16）の場合： 「腸腰筋」の短縮と「首」の回旋ストレスケアが必須項目となります。
