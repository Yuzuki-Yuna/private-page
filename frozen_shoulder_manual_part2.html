パターンC：下方拘縮優位型

重症度：中〜高 | 施術時間：60-70分

長期間の三角巾固定後、腕を下げたまま過ごす習慣がある患者に多い。下方関節包・腋窩陥凹部の拘縮により挙上が著しく制限される。

🔬 病態メカニズム：なぜ下方が拘縮するのか

【長期固定による下方組織の短縮】

骨折・脱臼後の三角巾固定、または腱板損傷後の安静により、腕を下垂位（体側につけた状態）で数週間〜数ヶ月過ごすと、下方関節包と腋窩陥凹部（脇の下の袋状の組織）が短縮します。

通常、腕を上げる際には下方関節包が伸張されますが、固定期間中は一度も伸張されないため、組織は「この長さで良い」と認識し、コラーゲン線維が短縮位で再構築されてしまいます。

【恐怖回避行動による悪循環】

固定解除後も、「腕を上げると痛い」という恐怖から、患者は腕を下げたまま過ごします。これにより下方関節包の短縮がさらに進行し、廃用性拘縮の悪循環に陥ります。

【肩峰下インピンジメントの発生】

下方関節包が拘縮すると、腕を上げる際に上腕骨頭が下方に滑り降りることができず、上方へ押し上げられます。これにより肩峰（肩の骨の出っ張り）と上腕骨頭が衝突（インピンジメント）し、激痛が走ります。

この痛みがさらに恐怖回避行動を強化し、拘縮が進行する悪循環となります。

🎯 鑑別診断：パターンCの確実な見分け方

視診 見た目で確認できるサイン
1. 立位姿勢観察
肩の下垂：患側の肩が健側より下がって見える（僧帽筋上部の弛緩）
腕の固定：腕を体側にぴったりつけて歩く。腕の振りがほとんどない
腋窩の狭小化：脇の下（腋窩）が狭く見える。腋窩陥凹部の癒着により
2. 動作観察（最重要）
挙上の著しい制限：腕を前方または側方から上げる動作が90度未満で止まる
肩全体が上がる代償動作：腕を上げようとすると、上腕骨ではなく肩甲骨全体（僧帽筋の収縮）が上がる。これを肩すくめ動作と呼ぶ
体幹側屈の代償：腕を上げる際に、体を反対側に傾けて代償する
触診 手で触って確認する拘縮
1. 腋窩陥凹部の触診（最重要）
  1. クライアント座位、腕をリラックスさせる
  2. 施術者は患側の脇の下に指を入れる
  3. パターンCの特徴：
    • 通常、脇の下はふんわりとした空間があるが、パターンCでは空間が消失している
    • 指を深く入れようとすると、硬い組織に阻まれる
    • 腋窩陥凹部（関節包の下部が袋状になっている部分）が癒着し、板状に硬くなっている
2. 下方関節包の触診
  1. クライアント座位、腕を軽く外転させる
  2. 施術者は上腕骨頭の下方に指を当てる
  3. パターンCの特徴：関節包が厚く硬い。健側と比較すると明らかに硬い
徒手検査 可動域テスト
1. 他動挙上テスト（最も重要）

実施方法：

  1. クライアント座位または仰臥位
  2. 施術者は一方の手で肩甲骨を固定（肩甲骨が動かないように上から押さえる）
  3. もう一方の手でクライアントの肘を支え、ゆっくりと前方挙上させる
  4. 健側も同様に行い、比較

パターンCの陽性所見：

  • 他動挙上が90度未満で硬い終末感（健側は通常180度）
  • 60-90度付近で、上腕骨頭が肩峰に突き上げられる感覚（インピンジメント）
  • 肩甲骨を固定すると、さらに制限が顕著になる（肩甲上腕リズムの破綻）
2. Neerインピンジメントテスト
  1. 施術者が一方の手で肩甲骨を固定
  2. もう一方の手でクライアントの腕を強制的に前方挙上
  3. 陽性所見：90度付近で激痛。肩峰下でのインピンジメント
3. Sulcus Sign（サルカスサイン）
  1. クライアント座位、腕をリラックスさせる
  2. 施術者がクライアントの腕を下方に軽く牽引
  3. 陽性所見：肩峰と上腕骨頭の間に溝（sulcus）ができる。下方関節包の弛緩を示唆
  4. 注意：パターンCでは通常陰性（溝ができない）。下方関節包が拘縮しているため

🔧 具体的な施術方法（ステップバイステップ）

施術の全体方針
パターンCでは、下方関節包と腋窩陥凹部の癒着剥離が最優先です。この領域はアクセスが困難で、技術的難易度が高いため、段階的に慎重にアプローチします。また、インピンジメント予防のため、肩甲骨周囲筋のバランス調整も重要です。

推奨施術時間：60-70分
体位変換：座位 → 仰臥位 → 側臥位

1温熱療法（10分）

クライアント座位。肩全体（特に腋窩周囲）にホットパック15分。腋窩陥凹部は深層にあるため、超音波療法（1MHz / 連続モード / 1.0W/cm²）を併用すると効果的。照射部位：腋窩前方・下方・後方の3箇所、各3-5分。

2肩甲骨周囲筋の準備リリース（10分）

準備 僧帽筋・菱形筋・前鋸筋のリリース

目的：肩甲骨の可動性を改善し、肩甲上腕リズムを正常化。下方関節包の施術前に、肩甲骨が自由に動ける状態を作る。

体位：座位または腹臥位

手順：
  1. 僧帽筋（3分）：首の付け根から肩甲骨上角まで、軽擦→揉捏。圧は5-6/10
  2. 菱形筋（3分）：肩甲骨の内側縁を親指で押圧。圧は6-7/10
  3. 前鋸筋（4分）：側臥位で、肩甲骨外側縁の深層にある前鋸筋をリリース。圧は5-6/10

3腋窩陥凹部の癒着剥離（15分）

⚠️ 重要な注意：腋窩には腋窩神経、腋窩動脈・静脈、リンパ節が密集しています。圧が強すぎると神経損傷、血管損傷、リンパ浮腫のリスクがあります。必ず軽圧から始め、クライアントの反応を常に確認してください。
最重要 腋窩陥凹部の段階的リリース

体位：仰臥位、腕を軽く外転（45-60度）、肘は軽く屈曲

手順：
  1. 腋窩の開放（3分）：
    • クライアントの腕を支え、腋窩を開く
    • 施術者は患側の脇の下に指（人差し指・中指・薬指の3本）を優しく入れる
    • 圧は3-4/10の非常に軽い圧から開始
    • クライアントに「痛いですか？しびれはありますか？」と確認
  2. 癒着部位の特定（2分）：
    • 腋窩内を指で触診し、硬く板状になっている部位を探す
    • 典型的な位置：腋窩の下方および前下方
    • この部位が腋窩陥凹部の癒着部
  3. 軽圧でのリリース（5分）：
    • 特定した癒着部位に、4-5/10の圧でゆっくりと圧を加える
    • 圧の方向：肩甲骨に向かって（内側方向）。決して外側（腋窩神経・血管方向）ではない
    • 持続圧を30-45秒加え、組織が緩むのを待つ
    • クライアントに深呼吸を促し、吐く息で圧を深める
    • 2-3箇所の癒着部位に順番にアプローチ
  4. 腕の他動運動との統合（5分）：
    • 腋窩陥凹部を圧迫したまま、もう一方の手でクライアントの肘を支える
    • ゆっくりと腕を挙上させる（前方挙上または側方挙上）
    • 圧迫により癒着が剥がれるタイミングで、可動域が急に広がることがある
    • 痛みが出る手前で止め、そのまま10-15秒保持
    • これを3-4回繰り返す

成功の指標：

  • 施術前より腋窩の空間が広がる
  • 板状の硬結が柔らかくなる
  • 他動挙上が10-20度改善

4下方関節包のストレッチ（15分）

Grade III 下方滑り（Inferior Glide）with Oscillation

目的：下方関節包を機械的に伸張。挙上可動域を改善し、インピンジメントを軽減。

体位：仰臥位、肩甲骨の下にタオルを入れて固定

手順：
  1. 開始肢位の設定：
    • クライアントの腕を可能な範囲で外転（通常60-90度）
    • 肘は伸展位
  2. グリップ：
    • 施術者は体幹側に立つ
    • ウェッブスペース（親指と人差し指の間）を上腕骨骨頭の直下に当てる
    • もう一方の手で肘を支える
  3. 下方滑り（10分）：
    • 圧の方向：患者の足方向（尾側）へ向けて、上腕骨頭を押し下げる
    • 圧の強さ：6-7/10
    • リズム：持続圧（30秒）と振動（オシレーション）を組み合わせる
      • 30秒持続圧を加える
      • その後、小刻みに上下に振動させる（振幅1-2cm、頻度2-3Hz）を10-15回
      • 10秒休憩
      • これを5-6回繰り返す
  4. 漸進的外転角度の増加：
    • 最初は60度外転位で下方滑り
    • 次に75度外転位で下方滑り
    • 最後に90度外転位（可能であれば）で下方滑り
    • 外転角度を増やすごとに、下方関節包がより効果的に伸張される

重要なポイント：

単に腕を下に引っ張るのではなく、上腕骨頭だけを下方に滑らせるイメージで行うことが重要です。肩甲骨は固定したまま、骨頭のみを動かします。

5大胸筋・広背筋のストレッチ（10分）

補助 挙上制限筋のリリース
大胸筋下部線維のストレッチ（5分）：
  1. クライアント仰臥位
  2. 施術者はクライアントの腕を外転・外旋させる
  3. 肘を支えながら、ゆっくりと後方へ倒す（水平伸展）
  4. 大胸筋が伸びるのを感じる位置で30秒保持
  5. 3-4回繰り返す
広背筋のストレッチ（5分）：
  1. クライアント座位
  2. 両手を頭の上で組ませる
  3. 施術者が後ろから両肘を支え、上方へ持ち上げる
  4. 広背筋が伸びるのを感じる位置で30秒保持
  5. 3-4回繰り返す

6自動介助運動（5分）

最終 Pulley Exercise（滑車運動）の模擬

目的：新たに獲得した可動域を、クライアント自身の力で確認。神経筋再教育。

手順：
  1. クライアント座位
  2. 健側の手で患側の手首を持つ
  3. 健側の力で患側の腕をゆっくりと前方挙上
  4. 施術前より挙がることを確認し、クライアントに実感してもらう
  5. 10回 × 2セット

🏠 ホームケア指導（パターンC専用）

毎日行うセルフエクササイズ（朝晩2回）
1. 壁登り運動（Wall Walking）
  1. 壁の前に立つ（約30cm離れる）
  2. 患側の指先を壁につける
  3. 指で壁を登るように、少しずつ腕を上げていく
  4. 痛みが出る手前まで上げたら、そのまま10秒保持
  5. ゆっくり降ろす
  6. 10回 × 3セット
2. テーブルスライド（Table Slide）
  1. テーブルの前に座る
  2. テーブルの上にタオルを置く
  3. 両手をタオルの上に乗せる
  4. お辞儀をするように体を前に倒しながら、タオルを前方へ滑らせる
  5. 脇が伸びる位置で10秒保持
  6. 10回 × 2セット
3. ゴムバンド・エクササイズ（挙上筋強化）

重要：パターンCでは、拘縮の改善だけでなく、挙上筋（前鋸筋、僧帽筋下部）の強化も必要です。

  1. ゴムバンド（セラバンド等）を用意
  2. 立位で、ゴムバンドを足で踏む
  3. 両端を持ち、腕を前方挙上
  4. ゆっくり10回 × 3セット
恐怖回避行動の克服（最重要）
パターンCの最大の問題は、「腕を上げると痛い」という恐怖から、腕を使わなくなることです。施術により可動域が改善しても、恐怖が残っていると再び拘縮します。

指導のポイント：
  • 「痛みが出る手前まで」腕を動かすことは安全であり、必要であることを説明
  • 日常生活で意識的に腕を使う習慣をつける（例：高い棚から物を取る、洗濯物を干す等）
  • 「使わないと固まる」という廃用性拘縮のメカニズムを説明

⚠️ 禁忌事項・医療機関紹介基準（パターンC）

パターンC特有の注意点：
  • 腱板断裂との鑑別が重要。Drop Arm Test陽性、夜間痛激烈な場合は、MRI検査を推奨
  • リンパ浮腫の既往（乳がん手術後等）がある場合、腋窩への施術は禁忌
  • 腋窩リンパ節腫脹がある場合、感染症・悪性腫瘍の可能性があり、医療機関へ即座に紹介

パターンD：全方向拘縮型（最重症）

重症度：最高 | 施術時間：75-90分

関節包全体が線維化し、すべての方向で可動域が制限される。パターンA・B・Cの要素がすべて混在。糖尿病合併例、長期放置例に多い。「真の五十肩」。

🔬 病態メカニズム：なぜ全方向が拘縮するのか

【関節包全体の線維化カスケード】

パターンDは、パターンA・B・Cのいずれかから始まり、放置または不適切な管理により全方向に波及したケースが多いです。

例：前方拘縮（パターンA）で始まる → 痛みで腕を動かさなくなる → 下方も拘縮（パターンC要素追加）→ 代償動作で後方も緊張（パターンB要素追加）→ 全方向拘縮

【糖尿病との関連（重要）】

糖尿病患者では、健常者の5倍の頻度で肩関節周囲炎が発症します。高血糖によりAGEs（Advanced Glycation End Products：終末糖化産物）が関節包に蓄積し、コラーゲン線維の架橋を異常に促進。結果、関節包全体が硬く縮んだ状態になります。

糖尿病合併例では、通常の肩関節周囲炎より2-3倍長く症状が持続し（平均30ヶ月 vs 15ヶ月）、改善も遅いです。

【炎症の持続化】

パターンDでは、拘縮だけでなく、関節包内の慢性炎症も持続していることが多いです。滑膜の増生、血管新生、炎症性サイトカイン（IL-1β、TNF-α）の産生が続き、痛みと拘縮の悪循環を形成します。

🎯 鑑別診断：パターンDの確実な見分け方

病歴聴取 重要な情報
症状の持続期間：6ヶ月以上続いている（通常の肩関節周囲炎は15-18ヶ月で自然寛解することが多い）
糖尿病の既往：糖尿病があるか、血糖値が高いと言われたことがあるか
夜間痛：夜間に激痛で目が覚める。睡眠が著しく障害される
ADL（日常生活動作）の著しい制限：衣服の着脱、入浴、整髪など、ほぼすべての動作が困難
徒手検査 全方向の可動域制限
Capsular Pattern（関節包パターン）の確認

肩関節周囲炎では、特定の順序で可動域制限が現れます。これをCapsular Patternと呼びます。

肩関節のCapsular Pattern：

外旋 > 外転 > 内旋 の順で制限される

測定方法：

  1. 他動外旋：仰臥位、肘90度屈曲。健側 vs 患側
    パターンDでは：患側が20度未満（健側は通常70-90度）
  2. 他動外転：仰臥位または座位。肩甲骨固定して測定
    パターンDでは：患側が60度未満（健側は通常180度）
  3. 他動内旋：腹臥位、肘90度屈曲
    パターンDでは：患側が30度未満（健側は通常70-90度）

パターンDの診断基準：

  • 外旋、外転、内旋のすべてが健側の50%以下に制限
  • かつ、制限の程度が 外旋 > 外転 > 内旋 の順
End Feel（終末感）の評価

各方向の可動域の最終点で、どのような感触があるかを評価します。

  • パターンDの特徴：すべての方向で硬い終末感（Firm / Hard End Feel）。まるで骨にぶつかるような、まったく遊びのない硬い感触。
  • 健常肩：軟らかい終末感（Soft End Feel）または弾力的終末感（Elastic End Feel）

📋 医学的に関与する組織（全方向）

組織 パターンDでの状態 触診・検査所見

🔧 具体的な施術方法（ステップバイステップ）

施術の全体方針
パターンDは最重症のため、1回の施術で劇的な改善は期待できません。長期戦を覚悟し、「優しく、じっくり、継続的に」が原則です。

最重要ポイント：
  • 強い刺激は絶対に禁物。防御性筋収縮を引き起こし、逆効果
  • 温熱療法と軽圧マッサージを重視
  • 週2-3回 × 3-6ヶ月の長期計画
  • 糖尿病合併例では、血糖コントロールの改善が必須（内科医と連携）
推奨施術時間：75-90分
体位変換：仰臥位 → 側臥位 → 腹臥位 → 座位

1温熱療法（最重要・20分）

深部温熱療法の徹底

目的：線維化した組織の粘弾性を高める。疼痛閾値を上げる。炎症性サイトカインの代謝を促進。

実施方法：
  1. ホットパック（15分）：
    • 肩全体（前方・後方・上方・腋窩）を覆うように当てる
    • 温度：40-45度
    • 低温やけど防止のため、必ずタオルを挟む
  2. 超音波療法（10分）：
    • パラメータ：1MHz / 連続モード / 1.0-1.5W/cm²
    • 照射部位：
      • 肩甲下筋（腋窩前方）：3分
      • 烏口上腕靭帯（烏口突起周辺）：2分
      • 棘下筋（肩甲棘下）：3分
      • 下方関節包（腋窩下方）：2分
    • 効果：表層のホットパックでは届かない深部組織（関節包、靭帯）を直接温められる

注意：超音波療法が利用できない場合は、ホットパックを20分に延長し、可能な限り深部まで温めます。

2全身リラクゼーション（15分）

準備 超軽圧での全身マッサージ

目的：パターンDの患者は、長期間の痛みにより全身が緊張しています。特に頸部・背部・対側肩の緊張が強い。まず全身をリラックスさせることで、患側肩の施術効果が高まります。

手順：
  1. 頸部・後頭部（5分）：仰臥位で、後頭部と頸部を優しく軽擦・揉捏。圧は3-4/10
  2. 背部全体（5分）：腹臥位または座位で、背中全体を軽擦。圧は3-4/10
  3. 対側肩（5分）：健側の肩も代償的に過緊張していることが多い。軽擦・揉捏でほぐす

3患側肩の段階的アプローチ（40分）

圧の強さの厳守（最重要）
パターンDでは、どの組織も極度に硬化しています。「硬いから強く押す」は絶対に禁物です。強い圧は防御性筋収縮を引き起こし、翌日さらに硬くなります。

圧の原則：すべての手技で5-6/10以下の軽〜中程度の圧。クライアントが「痛気持ちいい」を超えて「痛い」と感じたら即座に圧を弱めます。
Phase 1 表層筋の軽擦・揉捏（10分）
対象：
  • 大胸筋・三角筋（前方・側方）：仰臥位、圧4-5/10
  • 僧帽筋・広背筋（後方）：腹臥位または座位、圧4-5/10

手技：軽擦→揉捏→軽擦の順。ゆったりとしたリズムで。

Phase 2 中層筋のトリガーポイント軽圧（15分）
対象と手順：
  1. 小胸筋（5分）：
    • パターンAの手技を参照
    • ただし圧は4-5/10までに抑える
    • 持続時間も45-60秒に短縮
  2. 棘下筋（5分）：
    • パターンBの手技を参照
    • 圧は5-6/10まで
    • トリガーポイントは2-3箇所のみ
  3. 肩甲下筋（5分）：
    • パターンAの手技を参照
    • ただし、パターンDでは非常に硬く、深く指が入らないことが多い
    • 無理に深部を狙わず、触診できる範囲で軽圧（5/10）
Phase 3 腋窩陥凹部の最軽圧リリース（10分）
手順：
  1. パターンCの手技を参照
  2. ただし、パターンDでは癒着が非常に強固なため、1回の施術で剥離するのは困難
  3. 目標：「少しずつ柔らかくする」こと。1回で5mm指が深く入れば成功
  4. 圧は4-5/10の軽圧
  5. 持続時間：各ポイント30-45秒
Phase 4 烏口上腕靭帯の軽度横断摩擦（5分）
手順：
  1. パターンAの横断摩擦を参照
  2. ただし圧は5-6/10に抑える（パターンAでは7-8/10だが、パターンDでは軽め）
  3. 持続時間：各ポイント20-30秒

4関節モビライゼーション（15分）

Grade II 全方向への軽度モビライゼーション

注意：パターンDではGrade III（強いモビライゼーション）は禁忌。Grade II（軽度の動き）のみ。

実施する方向：
  1. 下方滑り（5分）：パターンCの手技。圧は5-6/10
  2. 後方滑り（5分）：パターンAの手技。圧は5-6/10
  3. 前方滑り（5分）：パターンBの手技。圧は5-6/10

リズム：すべて「2秒圧迫 → 2秒リリース」のリズミカルな動き。強い持続圧は避ける。

5アイシング（オプション・5分）

施術後、炎症の兆候（熱感・腫脹・夜間痛の増強）がある場合はアイシング10分。炎症がない場合は、再度ホットパック5分で終了し、組織を温かい状態で保つ。

🏠 ホームケア指導（パターンD専用）

パターンD専用ホームケアプログラム
1. 温熱療法（毎日2回、朝晩）

最重要：自宅でもできる限り肩を温めます。

  • 入浴時：シャワーで肩に温水を5-10分当てる
  • 就寝前：ホットパック（市販の温熱シート、カイロ等）を肩全体に20分
  • 注意：低温やけどに注意。必ず衣服の上から
2. 超軽度のストレッチ（毎日3回）

原則：「痛みが出る手前まで」。決して無理をしない。

  • 振り子運動（コッドマン体操）：重りは持たず、腕の重みだけで揺らす。前後・左右各30秒
  • テーブルスライド：パターンCの手技。ただし無理に遠くまで滑らせない。現在の可動域の範囲内で
3. 睡眠時のポジショニング（毎晩）

目的：夜間痛を軽減し、睡眠の質を確保。

  • 仰向け：患側の肘の下にクッション。肩の高さより肘を高くする
  • 側臥位（患側が上）：大きな抱き枕を抱き、患側の腕を抱き枕の上に乗せる。腕が下に落ちないよう、体幹と水平以上の高さを保つ
  • 患側下寝は絶対に避ける
4. テニスボール・セルフリリース（毎日1-2回）
  • 壁とのテニスボールを使い、烏口突起、棘下筋、小胸筋を軽圧（4-5/10）でセルフリリース
  • 各ポイント30-60秒
⚠️ パターンD専用の禁忌事項
  • 絶対に強い手技を行わない：パターンDは「優しく、じっくり」が絶対原則
  • 糖尿病合併例では必ず内科医と連携：血糖コントロール不良のまま施術しても改善は困難
  • 3ヶ月施術しても全く改善がない場合：MRI検査を推奨。腱板断裂、石灰沈着、関節リウマチ等の鑑別が必要
  • 夜間痛が激烈で睡眠が全く取れない場合：整形外科でのステロイド関節内注射を検討

施術者へのメッセージ

肩関節周囲炎は、施術者にとっても患者にとっても、根気と忍耐が求められる疾患です。特にパターンDでは、「早く治してあげたい」という気持ちから強い手技を行いがちですが、それは逆効果です。

「優しく、じっくり、継続的に」。この原則を守り、患者に寄り添い、長期的な視点で施術を行うことが、最も確実な改善への道です。

あなたの知識と技術が、多くの患者の人生の質を改善します。自信を持って、しかし謙虚に、施術に臨んでください。
